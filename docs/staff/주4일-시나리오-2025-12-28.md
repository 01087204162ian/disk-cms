## 3. 핵심 포인트

### 3.1 cycle_start_date
- **2025-12-28 (일요일)**에 설정했으므로
- **`cycle_start_date = 2025-12-29`** (다음 주 월요일)로 자동 설정

### 3.2 1주차 특수 상황
- **1주차 (2025-12-29 ~ 2026-01-04)**에 **1월 1일 공휴일** 포함
- 이 주는 **주 4일 근무가 해제**됨
- 모든 평일이 `is_off_day: false`로 표시됨
- 원래 휴무일이어도 공휴일 포함 주이므로 근무일로 처리

### 3.3 2주차부터 정상 적용
- **2주차 (2026-01-05 ~ 2026-01-11)**부터 정상적으로 주 4일 근무 적용
- 선택한 `base_off_day`가 휴무일로 표시됨

### 3.4 시프트 순환
- **사이클 0 (1-4주차)**: 선택한 `base_off_day` 고정
- **사이클 1 (5-8주차)**: 시프트 시작 (금→목→수→화→월)
